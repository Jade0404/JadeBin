⚙️ กลไกภายในถัง — Smart Sorting Mechanism

🔍 ภาพตัดขวาง — มองจากด้านข้างผ่ากลางถัง
⚠️ Reject Flap — ปกติ vs ตรวจพบถุงขยะ
✅ ปกติ — ขยะชิ้นเดียว
⚠️ ถุงขยะ — น้ำหนักผิดปกติ
📋 ลำดับการทำงานทั้งหมด
1IR Sensor ตรวจจับขยะตัดผ่านลำแสง → trigger ระบบ
2Reject Flap รอรับ — Load Cell ชั่งน้ำหนักทันที
3Landing Zone ขยะพัก ~0.3 วินาที — Inductive + Weight ส่งข้อมูล ESP32
4aถุง (น้ำหนัก > 400g) → Flap เอียง → คืนออกด้านข้าง → LED แดง → QR แจ้ง
4bโลหะ → Gate 1 เอียงซ้าย → ช่องโลหะ
4cเปียก/อินทรีย์ → Gate 2 เอียงซ้าย → ช่องอินทรีย์
4dแห้ง → Gate 2 เอียงขวา → ช่องแห้ง (รีไซเคิล+ทั่วไป)
5QR Code ขึ้นบนหน้าจอ + Firebase บันทึก + Community Dashboard
